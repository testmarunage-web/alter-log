# Hack Log (ジャーナリング2.0) 要件定義書

## 1. プロダクト概要
ビジネスパーソン向けの専属AIコーチングPWA（Progressive Web App）。
「ただ記録する」のではなく、AIからの能動的な問いかけに音声で答えることで、思考を強制的に引き出し、行動変容を促す。

## 2. 3つのコアバリュー（UXループ）
1. 着火（能動的問いかけ）: アプリを開くとAIから文脈に沿った問いが待機している。
2. 伴走（RAGによる深掘り）: 過去のログをベクトル化して記憶・検索し、本質的な課題を突く容赦ない壁打ち。
3. 価値化（週次レポート）: 1週間の対話から「美しいグラフ」と「鋭い総評」をNext.js上で自動生成し、成長を可視化。

## 3. ビジネスモデル・価格戦略
* 無料トライアル: 一切なし（登録＝即課金）。
* アーリーバード価格: リリース〜指定期限まで「月額3,000円」。この期間の登録者は解約しない限り一生涯3,000円。
* スタンダード価格: 期限終了後、自動で「月額5,000円」に切り替え。

## 4. コアUX・機能要件
* PWA: ホーム画面追加を必須とする独立したアプリ体験。
* 初期アンケート: 初回ログイン時に4問程度の選択式アンケートを実施し、AIの人格を固定する。
* 音声入力: PWA内のマイクボタン等の複雑な実装は避け、OS標準キーボードの音声入力を活用するUIにする。
* セッション制限UI: チャット画面上部にバッテリー型等のアイコンで「本日の残回数（例: 25/30）」を当表示。
* クリフハンガー: 毎日のセッション終了時、AIに「明日の予告テーマ」を語らせる。

## 5. リスクヘッジ要件（Stripe凍結対策）
* 解約の簡略化: アプリ設定画面に「Stripeカスタマーポータル」へのリンクを設置し、ユーザーが自ら1クリックで解約できるようにする。

## 6. 技術スタック
* Frontend/Backend: Next.js (Vercel)
* Database: Prisma + PostgreSQL
* AI/RAG: OpenAI API または Anthropic API + Embedding
* Payment: Stripe
